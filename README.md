## Prerequisites
- [hugo](https://gohugo.io/)
## Usage
#### Create new content
```
hugo new content <SECTIONNAME>/<FILENAME>.<FORMAT>
```
#### Run development server
```
hugo server --buildDrafts
```
#### Update theme
```
git submodule update --remote --merge
```
